# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this package, please send an email to Steven Smith at **steven@sd-smith.co.uk**.

All security vulnerabilities will be promptly addressed. Please do not disclose security-related issues publicly until a fix has been announced.

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.x     | Yes                |

## Response Timeline

- **Acknowledgement**: Within 48 hours of report
- **Assessment**: Within 1 week
- **Fix release**: As soon as possible, depending on severity
